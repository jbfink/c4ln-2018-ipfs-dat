a peek at two decentralized data sharing protocols: IPFS and dat
john fink, mcmaster university
ipfs
decentralized
permanent
p2p
Intended to be a *replacement* for http
Each ipfs entry gets a cryptographic *hash*
automatically deduped
Can be given intelligible names via a naming system
Has cheerful and easy Firefox and Chrome extension
dat
basically everything about ipfs *but*
Versioned (can change data and use same name)
Does not dupe data locally
is designed to be network agnostic
Let's experiment????
